Cuerpo-Documento


El cuerpo esta delimitado por las etiquetas <body> y </body>, y entre ellas estara el contenido de nuestra pagina web.


Establecer una imagen de fondo


Para ello se utiliza el atributo 'background' que nos permite poner una imagen como fondo de la pagina. Asi como se puede usar imagenes de fondo tambien puedes rellenar de fondo de un color con la propiedad 'background-color'. La propiedad 'background-color' permite el uso de 2 formatos: Mediante el nombre del color y RGB.




Establecer color de texto


Para establecer el color de primer plano para el texto se utiliza el atributo 'color'. Esta etiqueta acepta los dos formatos mencionados anterioremente, RGB y nombre

Inserccion de texto

Las etiquetas que definen los titulos de seccion son desde <h1> hasta <h6> en donde 'h1' es la de mayor tamano y 'h6' es la de menor tamano. La etiqueta que define el parrafo es <p> y </p>. Estas etiquetas aceptan ademas el atributo 'text-align', que permite la alineacion horizontal.


Resaltar caracteres


Para enfatizar una zona del texto se pueden usar las siguientes etiquetas:

Cite: Indica que el texto es una direccion postal.

Address: Indica que el texto es una cita de otro texto. Generalmente se utiliza con <blockquote>.

Q: Permite definir citas cortas.

Blockquote: Indica que el texto es un bloque sangrándolo por los dos lados y sangrándolo de los párrafos superiores e inferiores.

Em: Indica que nos interesa enfatizar el texto.

Strong: Indica un enfasis mayor al de 'em'.

I: Indica que el texto dee mostrarse en cursiva

U: Indica que el texto debe mostrarse subrayado

Strike: Indica que el texto debe mostrarse tachado

Sub: Indica que el texto debe mostrarse como un subindice

Sup: Indica que el texto debe mostrarse como un superindice

Mark: Indica que el texto debe mostrarse marcado y resaltado




Parrafos

La etiqueta <p></p> es la mas utilizada ya que permite definir los parrafos que forman el texto de un pagina. Esta etiqueta tiene el atributo 'text-align' que permite alinear horizontalmente. Los valores que puede tomar son: center, left, right y justify. si se desea introducie espacios entre parrados se debe insertar saltos de lineas mediante la etiqueta <br>


Listas

Con la etiqueta <ol></ol> es posible representar enumeraciones de elementos en formas de listas, dentro de estas se podra incluir cualquier elemento del lenguaje, inclsuo otra lista (lista anidada). En HTML existen tres tipos de listas:

Listas no ordenadas: La lista no esta numerada. <ul></ul>

Listas ordenadas: La lista esta numerada. <ol></ol>

Listas de definicion: Un conjunto de terminos y definiciones, similar a un diccionario<dl&</dl>

Lista no ordenada

No indica un orden o secuencia determinado. La etiqueta <ul> encierra todos los elementos de la lista y la etiqueta <li> cada una de las ocurrencias de la lista.

Lista ordenada

La etiqueta que nos permite crear esta lista es <ul>. Esta lista si presenta un orden ya sea alfabetico o numero, aunque no esta relacionado con el contenido. La etiqueta <li></li> encierra el contenido de la lista.

Lista de definiciones

Se utiliza para asociar un termino y la definicion del mismo. La etiqueta <dl></dl> encierra a la lista, la etiqueta <de></dt> encierra el termino a definir y la etiqueta <dd></dd> encierra la descripcion del termino
