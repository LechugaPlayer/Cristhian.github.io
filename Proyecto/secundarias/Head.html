Introduccion


A lo largo del tutorial usaremos el estandar de HTML 4.01 ya que es comptaible con la gran mayoria de navegadores actuales. Es importante especificar la version a usar pues se han hecho cambios a lo largo de la historia de HTML en donde muchas etiquetas han sido desaprobadas.


Estructura basica de un documento

Podras observar a continuacion que las instrucciones se encuentran encerradas entre los caracteres < y > como <html></html>. Esto ocurre con casi todas las etiquetas de html, en lineas generales inician con <etiqueta> y finaliza con </etiqueta>. Aquellas etiquetas que requieren sesr encerradas entre dos angulos son llamadas contenedoras y aquellas que no, vacias.




La eiqueta <br> realiza un salto de pagina y es una etiqueta vacia

Una pagina HTML tiene dos secciones muy definidas que vendrian a ser la cabecera(head) y el cuerpo(body)


La seccion 'head' siempre contiene informacion sobre el titulo de la pagina, el autor, etc. Esta informacion no sera visible en el navegador, salvo el titulo que se vera en la pestana.


En la seccion 'body' va escrito todo el contenido visible en la pagina Web


Caracteres, entidades y caracteres especiales


En algun momento se presentara un coflicto con el editor de texto al insertar un caracter distinto al ASCII. Para solventar este problema se pueden utilizar entidades de caracter que son interpretados por todas las plataformas: nombradas, numeradas y reservadas

Nombradas: Inician con el caracter '&', la abreviatura del caracter seguido de un punto y coma.
Numeradas: Inician con los caracteres &#, el numero de la posicion del caracter en la posicion ISO-Latin-1 seguido de un punto y coma.
Reservadas: Se usan exclusivamente para caracteres usados en eiquetas pero que nosotros no los pretendemos usar dentro de ninguna de ellas


Comentarios


Es un texto que ignora el navegadaor web, por tanto es utilizado para documentar el codigo ya que la unica manera de verlo es a traves de codigo fuente. Por tanto, esta funcionalidad esta pensada para programadores web ya que permite documentar el codigo en caso de que otra persona quiera modificarlo en un futuro.




Interpretacion de saltos de lineas y tabulaciones


El editor de texto no representa los saltos de lineas asi como las tabulaciones. La etiqueta usada para realizar un salto de linea es <br>.




Cabecera de un documento html


La cabecera figura entre las etiquetas <head> y </head>. En ella se incluiran definiciones generales como el titulo, autor,meta tags, etc.


Titulo del documento


Se coloca entre las etiquetas <title> y </title>. Nos premite definir el titulo que aparecera en la barra del navegador



Metadatos


Es una etiqueta vacia utilizada para definir el contenido de ciertas variables relacionadas con las paginas web, como el nombre del autor, la descripcion de la pagina, etc.



Author: Indica el nombre del autor del documento

Description: Description del sitio web

Editor: Indica el programa usado para hacer la pagina

Keysword: Palabras claves que permiten a alos buscadorees indexar su contenido
